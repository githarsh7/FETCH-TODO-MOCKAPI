# FETCH TODO MOCKAPI

A React-based Todo Application integrated with MockAPI for persistent task management. This project allows users to create, update, complete, and delete tasks while storing data permanently using a REST API. The application ensures that todos remain available even after refreshing the browser.

## Live Features

- Add new todo tasks
- Edit existing tasks
- Mark tasks as completed or not completed
- Delete tasks permanently
- Persistent data storage using MockAPI
- Axios-based API integration
- Responsive user interface
- Formula 1 themed modern UI

## Installation and Setup

### Clone the Repository

```bash
git clone https://github.com/githarsh7/FETCH-TODO-MOCKAPI.git
```

### Navigate to Project Folder

```bash
cd FETCH-TODO-MOCKAPI
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

---

## MockAPI Setup

1. Visit MockAPI:

```text
https://mockapi.io/
```

2. Create a new project

3. Create a resource named:

```text
todos
```

4. Copy your generated endpoint:

```text
https://your-project-id.mockapi.io/todos
```

5. Replace the API URL inside the project.

## CRUD Operations

The application performs the following API operations:

| Method | Purpose |
|--------|---------|
| GET | Fetch all todos |
| POST | Create new todo |
| PUT | Update existing todo |
| DELETE | Delete todo |
